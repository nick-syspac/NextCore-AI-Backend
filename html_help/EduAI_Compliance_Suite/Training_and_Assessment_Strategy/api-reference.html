Complete API documentation for TAS system

Base URL

https://api.nextcore.ai/api/tenants/{tenant}/tas/

All TAS endpoints are tenant-scoped. Replace {tenant} with your RTO's tenant identifier.

üîê Authentication

All API requests require authentication via Bearer token:

Authorization: Bearer YOUR_ACCESS_TOKEN

Templates API

Manage TAS template configurations

List Templates

GET /templates/

Retrieve all templates for your tenant.

Query Parameters

Parameter Type Description
template_type String Filter by type: general, trade, business, etc.
aqf_level String Filter by AQF level
is_active Boolean Filter by active status
search String Search in name, code, description

Response

    "id": 5,

    "template_type": "business",

    "description": "For Certificate III business qualifications",

    "gpt_prompts": { ... },
    "is_active": true,
    "created_at": "2024-01-15T10:00:00Z",

Get Template

GET /templates/{id}/

Webhook Events

Subscribe to webhook events for async operations:

Event Description Payload
tas.generation.started TAS generation started { job_id, template_id, qualification_code }
tas.generation.completed TAS generation completed { job_id, tas_id, duration_seconds }
tas.generation.failed TAS generation failed { job_id, error, failed_section }
tas.approved TAS document approved { tas_id, approved_by, approved_at }

SDK Examples

Python

from nextcore import NextCoreClient

client = NextCoreClient(
    tenant="acme-rto",
    api_key="your_api_key"
)

# List templates
templates = client.tas.templates.list(template_type="business")

# Create section
section = client.tas.sections.create(
    section_name="Qualification Overview",
    section_code="QUAL_OVERVIEW",
    content_type="rich_text"
)

# Generate TAS
job = client.tas.generate(
    template_id=5,
    qualification_code="BSB30120",
    qualification_title="Certificate III in Business",
    context={
        "delivery_mode": "blended",
        "duration": "52 weeks"
    }
)

# Poll for completion
tas = client.tas.wait_for_completion(job.job_id, timeout=300)

JavaScript/TypeScript

import { NextCoreClient } from '@nextcore/sdk';

const client = new NextCoreClient({
  tenant: 'acme-rto',
  apiKey: 'your_api_key'
});

// List templates
const templates = await client.tas.templates.list({
  templateType: 'business'
});

// Create section
const section = await client.tas.sections.create({
  sectionName: 'Qualification Overview',
  sectionCode: 'QUAL_OVERVIEW',
  contentType: 'rich_text'
});

// Generate TAS
const job = await client.tas.generate({
  templateId: 5,
  qualificationCode: 'BSB30120',
  qualificationTitle: 'Certificate III in Business',
  context: {
    deliveryMode: 'blended',
    duration: '52 weeks'
  }
});

// Poll for completion
const tas = await client.tas.waitForCompletion(job.jobId, { timeout: 300000 });
