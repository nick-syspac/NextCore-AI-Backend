Database Configuration

TASConversionSession Table

The conversion service uses a dedicated model to track conversion sessions:

# Table: tas_conversion_sessions
# 
# Key fields:
# - source_tas: ForeignKey to TAS (2015-based document)
# - target_tas: ForeignKey to TAS (2025-based document)
# - status: pending, analyzing, mapping, converting, validating, completed, failed
# - standards_mapping: JSONField with 2015‚Üí2025 mappings
# - conversion_changes: JSONField tracking all changes
# - compliance_report: JSONField with validation results
# - quality_score: Float (0-100) AI-assessed quality

Conversion Service Class

# tas/conversion_service.py

class TASConversionService:
    """
    Service for converting TAS from 2015 to 2025 Standards
    
    Workflow:
    1. Analyze source TAS (2015)
    2. Map standards (2015 ‚Üí 2025)
    3. Convert content with AI
    4. Validate compliance
    5. Create target TAS (2025)
    """
    
    def __init__(self, ai_model='gpt-4o'):
        self.ai_model = ai_model
        self.llm = self._initialize_llm()
        self.standards_mapper = StandardsMapper()
        self.content_transformer = ContentTransformer(self.llm)
        self.compliance_validator = ComplianceValidator()
    
    def create_conversion_session(
        self,
        source_tas: TAS,
        session_name: str,
        user,
        options: dict = None
    ) -> TASConversionSession:
        """Create new conversion session"""
        
    def execute_conversion(
        self,
        session: TASConversionSession
    ) -> TASConversionSession:
        """Execute full conversion workflow"""

AI Configuration for Conversion

Model Requirements

TAS Conversion requires advanced AI models due to complexity of standards mapping:

Task Recommended Model Min Context
Standards Mapping GPT-4o / Claude-3-Opus 32K tokens
Content Transformation GPT-4o / Claude-3-Sonnet 16K tokens

Load Mapping Data

# Create management command to load mappings
# File: tas/management/commands/load_standards_mappings.py

from django.core.management.base import BaseCommand
import json

class Command(BaseCommand):
    help = 'Load 2015‚Üí2025 standards mappings'
    
    def handle(self, *args, **options):
        with open('db/seeds/standards_mapping_2015_to_2025.json') as f:
            data = json.load(f)
        
        # Store in cache or database table
        from django.core.cache import cache
        cache.set('standards_mappings_2015_to_2025', data, timeout=None)
        
        self.stdout.write(
            self.style.SUCCESS(
                f'Loaded {len(data["mappings"])} mappings'
            )
        )

# Run the command
python manage.py load_standards_mappings

AI Token Limit Exceeded

# Increase token limits or use a model with larger context
# In .env:
CONVERSION_MAX_TOKENS=8000
CONVERSION_AI_MODEL=gpt-4o  # Has 128K context

Conversion Takes Too Long

# Enable parallel processing for sections
# In .env:
CONVERSION_BATCH_SIZE=10
ENABLE_ASYNC_CONVERSION=true

# Use Celery for background processing
celery -A control_plane worker --loglevel=info

Low Quality Score
